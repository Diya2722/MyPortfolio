## 🚀 Running Locally

This is a static site — no build step, no dependencies to install.

```bash
git clone https://github.com/Diya2722/MyPortfolio.git
cd MyPortfolio
```

Then simply open `index.html` in your browser, or for the best experience (live reload, no CORS quirks), serve it locally:

```bash
# Using VS Code's Live Server extension (recommended)
# Right-click index.html → "Open with Live Server"

# OR using Python
python -m http.server 5500
```

Visit `http://localhost:5500` in your browser.
